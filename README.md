## Features

- Scrapes book details from the website:
  - Title
  - Price
  - Rating
  - Book URL
- Saves data into **Excel (`books_data.xlsx`)** for easy analysis
- Handles missing data gracefully
- Easy to customize for multiple pages or categories

---

## Technologies Used

- **Python 3**  
- **Requests** – fetch web pages  
- **BeautifulSoup (bs4)** – parse HTML  
- **Pandas** – manage and export data  
- **OpenPyXL** – required by Pandas to write Excel files
