## Features

- Scrapes book details:
  - Title
  - Price
  - Rating
  - Book URL
- Saves data into **Excel (`books_data.xlsx`)** for easy analysis
- Handles missing data gracefully
- Supports scraping multiple pages
- Easy to customize for different categories or sites
